## 📱 **Mobile Support & WebView Integration**

This project is fully optimized for **Mobile WebView**! You can:

- ✅ Open directly in **mobile browsers** (Chrome, Safari, Firefox)
- ✅ Install as **PWA** (Progressive Web App) on Android/iOS
- ✅ Integrate with **Android WebView** using Java
- ✅ Fully **responsive UI** for all screen sizes
- ✅ Works on **tablets, phones, and desktops**

## ✨ **Features**

### 🔐 **Two-Level Authentication System**
- **Admin Panel:** Complete user & test management
- **User Panel:** View test results & print reports
- Secure session-based authentication
- Password hashing with MD5

### 👨‍⚕️ **User Management**
- Register users with full details (Name, National Code, Phone)
- Display user list with result registration
- Input validation (10-digit National Code, 11-digit Phone)
- Unique username, national code, and phone validation

### 🧬 **Medical Test Results Management**

| Test Category | Tests Included |
|---------------|----------------|
| **CBC** | WBC, RBC, Hemoglobin, Hematocrit, Platelets |
| **FBS** | Fasting Blood Sugar |
| **Lipid Profile** | Cholesterol, Triglyceride, HDL, LDL |
| **Kidney Function** | BUN, Creatinine |
| **Liver Enzymes** | ALT, AST, ALP, Bilirubin |
| **Thyroid** | TSH |
| **Vitamins** | Vitamin D |
| **Iron** | Iron, Ferritin |

### 🖨️ **Features**
- ✅ Print test results with optimized print styles
- ✅ View test history for each user
- ✅ Responsive design for all devices
- ✅ Modern UI with smooth animations
- ✅ Secure data handling with PDO
